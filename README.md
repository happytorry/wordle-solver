# Wordle Solver

A Python application to solve Wordle puzzles using word filtering and strategic guessing.

## Features

- Filters word list based on clues (green, yellow, gray letters)
- Suggests optimal guesses using letter frequency analysis
- Interactive mode to play along with Wordle
- Automatic solving mode

## Project Structure

- `wordle_solver.py` - Main entry point
- `solver.py` - Core solver logic
- `word_list.py` - Word list management
- `game.py` - Game logic

## Usage

```bash
python wordle_solver.py
```

## Requirements

- Python 3.6+
- No external dependencies
